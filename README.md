# 📊 E-Commerce Customer Behavior Analytics Dashboard

## 🔍 Overview

This project is a Power BI dashboard built to analyze customer behavior, sales performance, and business insights using an e-commerce dataset.

The dashboard helps in identifying trends, top-performing categories, and high-revenue locations.

---

## 🚀 Features

* KPI Cards: Total Revenue, Total Orders, Total Customers, Average Order Value
* Monthly Revenue Trend Analysis
* Top Revenue Generating Categories
* Revenue by Payment Method
* Customer Distribution by City (Map)
* Dynamic Insights using DAX measures

---

## 🛠 Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
